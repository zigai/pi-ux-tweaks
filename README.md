# Pi UI Tweaks

Collection of UI customization packages for Pi.

## Extensions

| Package                                                        | Description                                                           | Install                                   |
| -------------------------------------------------------------- | --------------------------------------------------------------------- | ----------------------------------------- |
| [`@zigai/pi-footer`](packages/pi-footer)                       | Custom footer with worked-for token speed widget.                     | `npm install @zigai/pi-footer`            |
| [`@zigai/pi-loader-time`](packages/pi-loader-time)             | Adds elapsed time to loader messages.                                 | `npm install @zigai/pi-loader-time`       |
| [`@zigai/pi-model-alias`](packages/pi-model-alias)             | Configures model aliases.                                             | `npm install @zigai/pi-model-alias`       |
| [`@zigai/pi-response-renderer`](packages/pi-response-renderer) | Personal response rendering tweaks.                                   | `npm install @zigai/pi-response-renderer` |
| [`@zigai/pi-submit-mode`](packages/pi-submit-mode)             | Customizes busy Enter and Alt+Enter submit behavior.                  | `npm install @zigai/pi-submit-mode`       |
| [`@zigai/pi-tree`](packages/pi-tree)                           | Customizes session tree display with timestamps and preview behavior. | `npm install @zigai/pi-tree`              |
